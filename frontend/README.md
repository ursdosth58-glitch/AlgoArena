# AlgoArena

AlgoArena is a full-stack competitive programming platform inspired by LeetCode and Codeforces.

## Features

* User Authentication (JWT)
* Problem Solving Interface
* Monaco Code Editor
* Run Code on Sample Test Cases
* Submit Code on Hidden Test Cases
* Contest System
* Dashboard
* Submission History
* Solved Problems Tracking

## Tech Stack

### Frontend

* React
* Vite
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Local Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
